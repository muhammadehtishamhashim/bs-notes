## Preliminaries

### 1. Alphabet ($\Sigma$)
A finite, non-empty set of symbols or characters.
* **Binary:** $\Sigma = \{0, 1\}$
* **English:** $\Sigma = \{a, b, c, \dots, z\}$
* **Special:** $\Sigma = \{+, -, \times, \div, =\}$

### 2. String ($w$)
A finite sequence of symbols chosen from a specific alphabet.
* **Example:** If $\Sigma = \{0, 1\}$, then $w = 0110$ is a string.
* **Empty String ($\epsilon$ or $\lambda$):** A string with zero occurrences of symbols. Length is 0.

### 3. Length of String ($|w|$)
The total number of symbols in a string.
* $|w| = \text{count of characters}$
* $|\epsilon| = 0$

### 4. Concatenation
Appending one string to the end of another.
* If $w_1 = abc$ and $w_2 = de$, then $w_1w_2 = abcde$.
* **Property:** $|w_1w_2| = |w_1| + |w_2|$

### 5. Language ($L$)
A set of strings formed using a specific alphabet, usually defined by a rule.
* **Example 1:** $L_1 = \{ \text{strings starting with 'a'} \}$ over $\Sigma = \{a, b\}$.
* **Example 2:** $L_2 = \{ \epsilon, 0, 00, 11, \dots \}$ (strings with pattern rules).

## Summary Comparison

| Feature | Alphabet ($\Sigma$) | String ($w$) | Language ($L$) |
| :--- | :--- | :--- | :--- |
| **Definition** | Non-empty finite **set** of symbols. | Finite **sequence** of symbols. | **Set** of strings. |
| **Structure** | Building blocks (atomic). | Combination of blocks (linear). | Collection of combinations. |
| **Cardinality** | Always Finite. | Finite Length. | Can be Finite or Infinite. |
| **Example** | $\{0, 1\}$ | $0101$ | $\{0, 00, 11, 000...\}$ |

## Kleene Operations

### Kleene Star ($\Sigma^*$)
The set of **all possible strings** (of any length) that can be formed from $\Sigma$, **including** the empty string.
* **Formula:** $\Sigma^* = \Sigma^0 \cup \Sigma^1 \cup \Sigma^2 \cup \dots$
* **Key Note:** Represents an infinite set.
* **Example:** $\Sigma = \{a\} \rightarrow \Sigma^* = \{\epsilon, a, aa, aaa, \dots\}$

### Kleene Plus ($\Sigma^+$)
The set of all possible strings **excluding** the empty string.
* **Formula:** $\Sigma^+ = \Sigma^1 \cup \Sigma^2 \cup \dots$
* **Relation:** $\Sigma^+ = \Sigma^* - \{\epsilon\}$
