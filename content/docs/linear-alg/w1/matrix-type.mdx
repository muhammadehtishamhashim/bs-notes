# Types of Matrices

The Grade 11 textbook classifies matrices into distinct types based on their order and the internal arrangement of their elements. Understanding these definitions is critical for matrix algebra.

1. **Row Matrix**: A matrix containing exactly one row ($1 \times n$).
2. **Column Matrix**: A matrix containing exactly one column ($m \times 1$).
3. **Rectangular Matrix**: A matrix where rows $\neq$ columns ($m \neq n$).
4. **Square Matrix**: A matrix where rows = columns ($m = n$).
5. **Diagonal Matrix**: A square matrix where all non-diagonal elements are exactly zero ($a_{ij} = 0$ for $i \neq j$), and at least one principal diagonal element is non-zero.
6. **Scalar Matrix**: A diagonal matrix where all diagonal elements are identical ($a_{ii} = k$).
7. **Identity (Unit) Matrix**: A scalar matrix where all diagonal elements are exactly $1$. Denoted by $I_n$.
8. **Null (Zero) Matrix**: A matrix of any order where every element is zero ($a_{ij} = 0$). Denoted by $O$.
9. **Upper Triangular Matrix**: A square matrix where all elements *below* the main diagonal are zero ($a_{ij} = 0$ for $i > j$).
10. **Lower Triangular Matrix**: A square matrix where all elements *above* the main diagonal are zero ($a_{ij} = 0$ for $i < j$).

## Conceptual Questions

### Question 1
"Every Identity Matrix is a Scalar Matrix, but every Scalar Matrix is not necessarily an Identity Matrix." Verify this textbook statement.

**Solution:**
This is **True**. A scalar matrix requires the diagonal elements to be identical constants $k$ and non-diagonals to be $0$. An Identity matrix perfectly satisfies this condition where the constant is $k = 1$, making it a special subset of a scalar matrix. However, a scalar matrix where $k = 5$ is not an identity matrix, proving the reverse statement is false.

### Question 2
If a matrix $A$ is strictly classified as both an Upper Triangular AND Lower Triangular matrix simultaneously, what specific type of matrix must it be?

**Solution:**
The upper triangular condition implies $a_{ij} = 0$ for $i > j$ (everything below the diagonal is zero). The lower triangular condition implies $a_{ij} = 0$ for $i < j$ (everything above the diagonal is zero). Combining these means $a_{ij} = 0$ for all $i \neq j$. A square matrix where all non-diagonal elements are zero is defined as a **Diagonal Matrix**.

### Question 3
Is it possible for a Null (Zero) matrix to also be classified as a Scalar Matrix? Explain your reasoning.

**Solution:**
Yes, but only if the Null Matrix is a **square matrix**. A scalar matrix must be a square matrix where all diagonal elements are identical and non-diagonal elements are zero. A square null matrix satisfies this with the constant $k = 0$. However, a rectangular null matrix cannot be a scalar matrix.

## Exercise Questions

### Question 1
Identify the order and specific type of matrix $A$.
$$A = \begin{bmatrix} 7 & -2 & 3 \end{bmatrix}$$

**Solution:**
The matrix has $m=1$ row and $n=3$ columns. Its order is $1 \times 3$. Because it possesses exactly one row, the textbook classifies this as a **Row Matrix**.

### Question 2
Analyze the structural properties of matrix $B$ and classify it fully based on the textbook rules.
$$B = \begin{bmatrix} 4 & 0 & 0 \\ 0 & 4 & 0 \\ 0 & 0 & 4 \end{bmatrix}$$

**Solution:**
1. The order is $3 \times 3$, so it is a **Square Matrix**.
2. All non-diagonal elements are zero, making it a **Diagonal Matrix**.
3. All principal diagonal elements are exactly identical (all are $4$).
According to textbook definitions, $B$ is definitively a **Scalar Matrix**.

### Question 3
Find the values of $x$ and $y$ if the given matrix is an Identity Matrix ($I_2$).
$$M = \begin{bmatrix} 1 & x-5 \\ 2y+4 & 1 \end{bmatrix}$$

**Solution:**
By definition, a $2 \times 2$ identity matrix must have zeros on the non-diagonal positions. Equating corresponding non-diagonal elements to zero:
- $x - 5 = 0 \implies x = 5$
- $2y + 4 = 0 \implies 2y = -4 \implies y = -2$

### Question 4
Construct the diagonal matrix represented compactly as $D = \text{diag}(8, -3, 0)$.

**Solution:**
The notation $\text{diag}(x, y, z)$ is used in textbooks to compactly write a square matrix where $x, y, z$ form the main diagonal from top-left to bottom-right, and all other entries are zero.
$$D = \begin{bmatrix} 8 & 0 & 0 \\ 0 & -3 & 0 \\ 0 & 0 & 0 \end{bmatrix}$$

### Question 5
Classify the following matrix $U$.
$$U = \begin{bmatrix} 2 & 5 & 1 \\ 0 & 9 & 4 \\ 0 & 0 & 7 \end{bmatrix}$$

**Solution:**
Notice that all elements strictly below the main diagonal are zero ($u_{21}=0, u_{31}=0, u_{32}=0$). Therefore, this is an **Upper Triangular Matrix**.

### Question 6
Find $a, b, c$ such that $L$ is a Lower Triangular Matrix.
$$L = \begin{bmatrix} 5 & a-1 & b+2 \\ -3 & 4 & c-3 \\ 7 & 8 & 9 \end{bmatrix}$$

**Solution:**
For a lower triangular matrix, all elements strictly above the main diagonal must be zero. We extract those entries and equate them to zero:
- $l_{12} = a - 1 = 0 \implies a = 1$
- $l_{13} = b + 2 = 0 \implies b = -2$
- $l_{23} = c - 3 = 0 \implies c = 3$

### Question 7
Classify the following matrix. Is it a column matrix or a row matrix?
$$V = \begin{bmatrix} 8 \\ -2 \\ 5 \\ 0 \end{bmatrix}$$

**Solution:**
The matrix has $4$ rows and $1$ column. Its order is $4 \times 1$. Because it consists of a single vertical column, the textbook classifies it as a **Column Matrix**.

### Question 8
Given that $D$ is a diagonal matrix, find the values of variables $x$ and $y$.
$$D = \begin{bmatrix} 5 & x - 2 \\ y + 1 & 8 \end{bmatrix}$$

**Solution:**
By definition, a diagonal matrix must have zeros in all non-diagonal positions. 
- $x - 2 = 0 \implies x = 2$
- $y + 1 = 0 \implies y = -1$

### Question 9
Construct a $3 \times 3$ scalar matrix given that the scalar constant is $k = -5$.

**Solution:**
A $3 \times 3$ scalar matrix has the constant $k$ on the main diagonal and zeros everywhere else.
$$S = \begin{bmatrix} -5 & 0 & 0 \\ 0 & -5 & 0 \\ 0 & 0 & -5 \end{bmatrix}$$

### Question 10
Classify the matrix $O$ fully based on its dimensions and elements.
$$O = \begin{bmatrix} 0 & 0 & 0 \\ 0 & 0 & 0 \end{bmatrix}$$

**Solution:**
The order of the matrix is $2 \times 3$, making it a rectangular matrix. Since every single element inside the matrix is $0$, it is strictly classified as a $2 \times 3$ **Null (or Zero) Matrix**.
